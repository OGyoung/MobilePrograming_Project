## 메인 액티비티 외의 것
1. 이름, 학과, 취미 등을 텍스트와 이미지 등으로 알려준다.
2. 간단한 자신의 일기장을 만든다.
3. 아날로그 시계와 디지털 시계를 보여준다.
4. 본인이 좋아하는 노래 한곡을 들을 수 있게한다.
